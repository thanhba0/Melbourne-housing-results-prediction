Thư viện:
scikit-learn : gọi các thuật toán máy học.
pandas: đọc dữ liệu.
numpy: xử lý dữ liệu.
time: tính toán thời gian chạy.
os: kiểm tra các đường dẫn, tạo thư mục.
collections: xử dụng Counter.
matplotlib: vẽ hình ảnh, trực quan hóa dữ liệu.
tkinter: thiết kế giao diện.

Cách biên dịch:
chạy file main.py trong thư mục source.

Các chức năng:

Choose file: chọn file dữ liệu để thực hiện train/test các thuật toán.
Load preprocessed data: tải dữ liệu đã tiền xử lý trước đó để test các thuật toán.

SVM : chọn thuật toán SVM.
Decision Tree: chọn thuật toán Decision Tree.
Logistic Regression: chọn thuật toán Logistic Regression.
Random Forest: chọn thuật toán Random Forest.
Naive Bayes : chọn thuật toán Naive Bayes.

Choose file save algorithm: chọn file lưu của thuật toán, lưu ý chọn thuật toán trước khi chọn file lưu.

Preprecess data: thực hiện các bước tiền xử lý dữ liệu.

Train algorithm: huấn luyện thuật toán, nếu không chọn "Choose file save algorithm" trước đó.
Evaluation algorithm: Đánh giá thuật toán trên tập test.
Save algorithm: lưu lại thuật toán.
Choose file and predict: chọn file cần dự đoán loại nhà, file label sẽ được lưu tại thư mục của file cần dự đoán với dạng *.txt
Predict sample: dự đoán 1 mẫu
